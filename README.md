# Приложение для работы с API RandomDataTools

## Описание проекта

Веб-приложение для загрузки, хранения и отображения информации о пользователях из внешнего API [RandomDataTools](https://randomdatatools.ru/developers/). 
Приложение позволяет загружать пользователей из API, просматривать их в таблице с пагинацией, а также смотреть детальную информацию 
о конкретном или случайном пользователе.

## Функциональность

- **Главная страница** - таблица со всеми пользователями из базы данных (пагинация по 20 записей)
- **Загрузка пользователей** - возможность вручную загрузить дополнительных пользователей из API
- **Страница пользователя** - детальная информация о конкретном пользователе (`/user/{id}`)
- **Случайный пользователь** - страница, показывающая случайного пользователя при каждом обновлении (`/random`)
- **Автоматическая загрузка** - при старте сервера автоматически загружается 100 пользователей

## Технологический стек

### FastAPI
**Выбор обоснован следующими причинами:**
- Высокая производительность (асинхронность из коробки)
- Автоматическая генерация OpenAPI документации (`/docs`)
- Простота разработки и минимальный boilerplate-код
- Встроенная валидация данных через Pydantic
- Отличная поддержка dependency injection

### SQLite
**Выбор обоснован следующими причинами:**
- Не требует отдельного сервера (embedded база данных)
- Идеально подходит для небольших проектов и прототипирования
- Простота настройки и использования
- Поддерживается практически всеми ORM
- Отличная производительность при небольших объемах данных (до 100k записей)

### SQLAlchemy (ORM)
**Выбор обоснован следующими причинами:**
- Самый популярный ORM в Python экосистеме
- Поддерживает множество баз данных (легко переключиться на PostgreSQL)
- Мощный API для построения сложных запросов
- Автоматическая обработка транзакций

### Jinja2 Templates
**Выбор обоснован следующими причинами:**
- Простой и интуитивный синтаксис
- Безопасное экранирование данных
- Возможность наследования шаблонов

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/9898d/random-users-app
cd random_users_app
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск приложения
```bash
uvicorn main:app --reload
```

### 5. Открыть в браузере
```
http://localhost:8000
```

## API Endpoints
| Метод | Эндпоинт | Описание |
| ----- | -------- | -------- |
| GET | `/` | Главная страница с таблицей пользователей |
| GET | `/api/users` | API для получения списка пользователей (пагинация) |
| POST | `/api/load_users` | Загрузка пользователей из внешнего API |
| GET | `/user/{id}` | Страница конкретного пользователя |
| GET | `/random` | Страница случайного пользователя |
| GET | `/users/count` | Получение количества пользователей в БД |
